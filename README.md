# Tic Tac Toe

A Tic Tac Toe game built with Vue 3, Vite, Tailwind CSS, and Motion.  
The player is **X** and plays against an AI **O**, infused by a minimax algorithm.  
X and O are rendered as animated SVGs with Motion (yes copy paste SVG's from example on their website).

Thomas

## Play! :joystick:

https://atictactoe.vercel.app

## Tech stack

- Vue 3 (script setup)
- Vite
- Tailwind CSS
- Motion (Vue bindings for Framer Motion-style animations)
- Atomic Design

## Getting started

Install dependencies:

```bash
npm install
```
